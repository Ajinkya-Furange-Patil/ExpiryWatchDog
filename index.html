🧪 Demo Use Cases

✅ Manual Tracking ➜ Expiry Table

Users can manually add products with expiry dates. The table then tracks time until expiry and gives visual alerts.

💥 Damaged Product Labels

Even if the expiry label is faded or torn, the system allows recognition using the QR code or barcode still present on the package.

📷 Upcoming: QR/Barcode Expiry Detection

  • Scan barcode/QR via camera
  • System fetches expiry info from database
  • Displays date with alert if needed
